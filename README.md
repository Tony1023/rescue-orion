# Getting started
Run
```
$ yarn install
```
or
```
$ npm install
```

Go to src/metadata and run
```
$ python gen.py
```

Then go back to root directory and run
```
$ yarn start
```
or
```
$ npm start
```
to start application on localhost:3000
